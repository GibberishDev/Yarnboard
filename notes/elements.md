<details>
<summary><h1>Descriptions</h1></summary>
<ul>

<li>
<h2> Text element </h2>
Text element is a simple floating text. Has <b>NO</b> pin connection.
</li>

<li>
<h2> Label element </h2>
Label is more advanced text element with a connection node and decorations
</li>

<li>
<h2> Picture element </h2>
Simple picture element. Displays images
</li>

<li>
<h2> Drawing element </h2>
Canvas for drawing and simple shapes
</li>

<li>
<h2> Pin </h2>
Pin is used for rerouting yarn connections
</li>

</ul>
